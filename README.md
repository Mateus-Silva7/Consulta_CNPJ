# Consulta CNPJ 🔍

Aplicação web para consulta de dados empresariais brasileiros a partir do CNPJ.

## 📚 Sobre o Projeto

O objetivo deste projeto foi criar uma aplicação web que permite consultar informações de empresas brasileiras a partir do CNPJ, utilizando a BrasilAPI como fonte de dados.

A aplicação exibe os dados retornados de forma organizada e permite exportar os resultados para PDF ou Excel, facilitando o uso das informações consultadas.

## 🛠️ Tecnologias Utilizadas

HTML5
CSS3
JavaScript
BrasilAPI
jsPDF
SheetJS

## 🎯 Conceitos Praticados

consumo de API pública com JavaScript
manipulação e exibição dinâmica de dados no DOM
exportação de dados para PDF com jsPDF
exportação de dados para Excel com SheetJS
layout responsivo
organização de código HTML, CSS e JavaScript

## 📱 Responsividade

O layout se adapta para diferentes tamanhos de tela, garantindo uma boa experiência de navegação em computadores, tablets e celulares.
